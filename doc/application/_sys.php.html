uses (line 133)

Uses is a pretty wrapper around include and require. Gives everything a more namespace vibe.

The first component of the uses, is one of the following path specifiers:

  • app Application path
  • sys The system path
  • system The system path
  • vendor The vendor path
  • model The path to the app's models
  • channel The path to the app's data channels
  • control Path to either the system controls or the app controls (Will try the system controls first, then the app controls)
For example, to load the Controller class which is in /sys/app/controller.php you would do the following:

  1.  uses('system.app.controller');

Can support wildcards:

  1.  uses('system.data.validators.*');
